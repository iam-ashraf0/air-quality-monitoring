# air-quality-monitoring
Low-cost Raspberry Pi system monitors air quality with real-time dashboard.

🌍 Air Quality Monitoring System

📌 Project Overview

This project is a low-cost Air Quality Monitoring System built using Raspberry Pi and sensors.
It measures air pollution levels and shows real-time data on a web dashboard.

The system helps users understand air quality and take steps for a healthier environment.

---

⚙️ Features

- 🌫️ Detects pollutants like PM2.5, PM10, CO₂, CO, NH₃, NOₓ, VOCs
- 📊 Real-time air quality monitoring
- 💾 Stores data in MySQL database
- 📈 Web dashboard with charts & graphs
- 📅 Historical data analysis
- 🔐 Admin login & report generation

---

🛠️ Technologies Used

Hardware:

- Raspberry Pi 3
- Air quality sensors
- MCP3008 ADC

Software:

- Python
- Flask
- MySQL
- HTML, CSS, Bootstrap
- Chart.js

---

🧠 How It Works

1. Sensors collect air quality data
2. Raspberry Pi processes the data
3. Data is stored in MySQL database
4. Flask backend sends data to dashboard
5. Users view real-time data in browser

---

📊 Dashboard Features

- Real-time AQI display
- Pollution level graphs
- AQI trend analysis
- 24-hour statistics
- Health recommendations
- Manual data entry option

📁 Database

Main table: "sensor_readings"

Stores:
